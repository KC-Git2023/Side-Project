Compile program:
cd to the upper directory of ticTacToe source
e.g. "C:\Users\...\Desktop\ticTacToe\source"

Then run the following command
javac ticTacToe/*.java -d "../classes"

Run the program
cd to directory of class folder
e.g. "C:\Users\...\Desktop\ticTacToe\classes"
or "cd ..\classes"

Then run the following command
java ticTacToe.ticTacToe6


Program Entry Point
ticTacToe6
